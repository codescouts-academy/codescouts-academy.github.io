Descripción

Teniendo en cuenta que React es una de las librerías más utilizadas en el mundo del Frontend, es imprescindible que tu equipo la domine a la perfección, en este curso enseáremos todo lo que necesitan para realizar una aplicación entera con React, aplicando excelentes prácticas. Compromiso de CodeScouts 🤞

Contenido

  • Integre TypeScript con las aplicaciones React
  • Reaccionar a props y estado usando interfaces de TypeScript
  • Stateless & Statefull
  • React Router router
  • Global state
  • Reaccionar el renderizado condicional
  • Manejo de componentes principales y secundarios con Typecript
  • Test de componentes
  • Utilice la interfaz, la interacción del usuario y las pruebas de código asíncrono
  • Functional component
  • React hooks
  • Custom hooks
  • Test de hooks

Valor del curso

  • Aprender a construir aplicaciones react con typescript
  • Reactividad en el frontend
  • Modularidad y componetización
  • Buenas prácticas de desarrollo front
  • Test de los componentes de react
  • Reusabilidad de código

Metodología

La parte más difícil de un curso es poner en práctica lo aprendido, por eso en este curso los alumnos pasan la mayor parte del tiempo programando, dedicaremos un 80% de práctica y un 20% de teoría.

El curso esta dividido en módulos y cada módulo consta de parte teórica y parte práctica.

La parte práctica se organiza trabajando en pareja, donde se fomenta un entorno comunicativo mediante ejercicios cortos. Haremos diferentes katas de programación para que los alumnos puedan practicar cada concepto que vamos aprendiendo.

Trabajaremos con mob programming

Requisitos

Este es un curso para personas con un mínimo de conocimientos de programación en Javascript y conceptos de Frontend.
